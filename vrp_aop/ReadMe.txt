standard aop with 5 options

if adding another location look through the server.lua and cfg.lua
there a commented section to each option to add a new entry for all catagories
just search for "new entry"

can configure in cfg file

perms 		- set perms for admins or default/basic players
entity 		- change blimp and color. link at top of cfg file
vote		- number of votes needed to set new aop via voting
auto		- enable or disable auto aop select
refresh		- time till auto aop selects new location
